DREADBOT V6 est un bot Discord dédié à la communauté ciné **BigscreenFR**.  
Il gère la programmation des soirées ciné, les défis interactifs, les GIFs de réaction,
les sondages films, et bien plus — propulsé par l'IA Groq/Gemini.

## ⚙️ Installation

### Prérequis
- Python 3.11+
- Token Discord Bot
- Clé API TMDB
- Token GitHub (pour GitHub Pages)
- Clé API Groq (pour les fonctions IA)

### Lancement
```bash
pip install -r requirements.txt
python bot.py
```

### Configuration (`config.py`)
```python
BOT_TOKEN          = "ton_token_discord"
TMDB_API_KEY       = "ta_cle_tmdb"
GITHUB_TOKEN       = "ton_token_github"
GROQ_API_KEY       = "ta_cle_groq"
SEANCES_CHANNEL_ID = 123456789   # ID du salon séances
OWNER_ID           = 123456789   # Ton ID Discord
```
