Synopsis Foireux
/fun → Défis → Synopsis

L'IA génère un synopsis complètement raté, absurde ou incohérent d'un film tiré de la base TMDB. Le bot choisit un film au hasard selon le genre sélectionné et réécrit son histoire… à sa manière délirante. Chaque synopsis est unique.

Note Mystère
/fun → Défis → Note

Le bot affiche une note (ex : 7.4/10) pour un film. Vraie ou inventée par l'IA ? Les membres votent en cliquant sur les boutons de réaction. Parfait pour tester la culture ciné du salon — personne ne sait si c'est réel.

Qui est-ce ?
/fun → Défis → Qui

Des indices sur un acteur ou personnage sont révélés progressivement. À chaque indice dévoilé, les membres tentent leur réponse via le bouton de réponse. Plus on attend, plus les indices se précisent — mais les points potentiels diminuent !

🤖 IA📢 Public

Trouve ton Film
/fun → Défis → Trouve

5 indices TMDB révélés un à un : genre, année, acteur principal, réalisateur, pays de production… À chaque indice, un bouton permet de tenter sa chance. Suspense garanti jusqu'à la révélation finale ! Source : API TMDB.

📢 Public🎲 TMDB

L'IA imagine une collision absurde entre deux univers de films totalement différents. Un pitch improbable, une histoire folle générée à la volée. Jamais deux fois pareil — le résultat dépend entièrement de l'humeur de l'IA ce jour-là.

Configuration des personnages IA — Réservé Owner uniquement

👑 N0 — Accès exclusif ✅ 🛡️ N2 — ❌ Fermé 🎬 N3 — ❌ Fermé 👤 N4 — ❌ Fermé
👑 /equipe → Prompts IA — Owner Only
La gestion des personnages IA est réservée exclusivement à Dreadlocktopus. Accessible via /superadmin → page 2 → 👥 Équipe IA. Cette interface permet de configurer des personnages IA avec des personnalités distinctes qui répondent différemment selon l'interlocuteur.
🎭
Gestion des Personnages IA
/superadmin → Équipe IA

Activez ou désactivez les personnages IA individuellement (jusqu'à 3 personnages configurés). Chaque personnage peut avoir des tons distincts : enthousiaste, délirant, poétique… Le mode peut être "aléatoire" (l'IA alterne les tons) ou "unique" (toujours le même).

👑 Owner🆕 V7
🔑
Accès par Rôle par Personnage
/superadmin → Équipe IA → Accès

Définissez quels rôles (N1, N2, N3, N4) ont accès à chaque personnage IA individuellement. Un personnage peut être activé pour N2 mais pas pour N3. Le Owner (N0) a toujours accès à tous les personnages, quels que soient les réglages.

👑 Owner🆕 V7
✍️
Prompt IA Libre
/superadmin → Équipe IA → Prompt

Envoyez un prompt libre à l'IA pour enrichir les bases de données ciné, générer du contenu de test ou tester un nouveau type de réponse. C'est l'interface de développement directe avec l'IA — puissant et sans garde-fou. Réservé au créateur du bot.

Antispam 🆕 V7

Protection automatique du serveur — Configurable et utilisable Owner uniquement

👑 N0 — Config & Usage ✅ 🛡️ N2 — ❌ Pas d'accès 🎬 N3 — ❌ Pas d'accès 👤 N4 — ❌ Pas d'accès
👑 /Antispam — Réservé Owner
Le module Antispam est entièrement configurable et utilisable par Dreadlocktopus uniquement. Il surveille en temps réel les messages du serveur et applique des sanctions automatiques selon les règles définies. Accessible via le panneau SuperAdmin intégré.
🔍
Détections Configurables
SuperAdmin → Antispam → Détections

Activez ou désactivez chaque type de détection indépendamment : Rafale de messages (trop de messages en peu de temps), Liens d'arnaque (URLs malveillantes), Compte trop récent (nouveaux comptes suspects), Absence d'avatar (profils vides souvent bots).

👑 Owner🆕 V7
⚖️
Sanctions Automatiques
SuperAdmin → Antispam → Sanctions

Choisissez la sanction appliquée automatiquement : Timeout (silence temporaire configurable en minutes), Kick (expulsion du serveur), Ban (bannissement). L'option de suppression automatique des messages détectés est également configurable.

👑 Owner🆕 V7
📊
Seuils & Paramètres
SuperAdmin → Antispam → Seuils

Configurez les seuils de déclenchement : nombre de messages par fenêtre de temps, durée de la fenêtre d'observation, âge minimum d'un compte (en jours) pour déclencher une alerte. Tous ces paramètres sont ajustables via un formulaire interactif.

👑 Owner🆕 V7
🛡️
Whitelist & Canal de Log
SuperAdmin → Antispam → Alertes

Configurez le salon de logs où les alertes antispam sont postées (avec détail du membre, raison de détection et actions effectuées). Un système de whitelist protège les membres de confiance des fausses détections — les rôles whitelistés ignorent les règles.

Éphémère vs. Public

Comprendre la visibilité des réponses du bot

👻 Éphémère

La réponse du bot est visible uniquement par vous. Personne d'autre ne la voit. Elle disparaît automatiquement après quelques minutes ou si vous rechargez Discord. Idéal pour les actions de gestion ou les infos confidentielles.

  • Panneau /admin
  • /diffusion programmer & gestion
  • /aide (résultats personnalisés)
  • Commandes de configuration SuperAdmin

📢 Réponse Publique

La réponse du bot s'affiche dans le salon, visible de tous. Essentiel pour les fonctions fun, les GIFs, les votes et tout ce qui vise à animer la communauté. Pensez à l'utiliser dans #Spam-The-Bot pendant les tests.

  • Tous les défis ciné (/fun)
  • GIFs de réaction
  • Votes et duels
  • Gaming, Média, Fun Facts
  • /calendrier, /liensutiles
⚙️
Comportement selon le niveau — La visibilité peut varier selon votre rôle. L'important est de tester les commandes dans #Spam-The-Bot afin de comprendre le comportement sans impacter les autres salons pendant la bêta.
